Also during this summer, I taught yoga & meditation to middle schoolers and high schoolers as part of a summer camp. I was able to connect with them socially and spritually. Not only did this make a positive impact on their wellbeing, but I also felt refreshed after my lessons.

NASA Space Apps Challenge

In October 2019 I attended my first ever hackathon! Created by NASA, the objective was to design a web application that would help solve any global challenge. My group came up with an application called SolSim, an educational and interactive solar system simulator in which a user can input parameters and the website will produce the desired system.

CS110 Global Sequence Alignment Project

In the Introduction to Programming in Python course I'm taking this Fall 2019, we had a second project in which we were assigned to calculate edit distance using dynamic programming as well as recover the alignment. It was definitely one of the hardest projects we have had to do by far, and it took a lot of effort and research to complete it.
